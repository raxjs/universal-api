```bash

```

## 使用

```js
import { createElement, useEffect, Fragment } from 'rax';
import { getApp } from '@uni/application';

function App() {
  useEffect(() => {
    const app = getApp();
  }, []);

  return (<>
    <View></View>
  </>)
}
```
也可以从大包中引入：

```js
import { application } from '@uni/apis';
```
