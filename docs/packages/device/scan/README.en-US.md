## Install

```bash
$ npm install @uni/scan --save
```
or
```bash
$ npm install @uni/apis --save
```
## 示例

```javascript
import scan from '@uni/scan';

scan({
  scanType: ['qrCode','barCode'],
  success (res) {
    console.log(res.result)
  },
  fail (res) {
    console.log(res)
  }
});

// promise
scan({
  scanType: ['qrCode','barCode'],
}).then(res => {
  console.log(res.result)
});

```


You can also import from big package:
```js
import { scan } from '@uni/apis';
```
