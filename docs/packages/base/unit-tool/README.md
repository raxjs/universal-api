## 安装

```bash
$ npm install @uni/unit-tool --save
```
## 示例

```js
import { px2rpx, rpx2px } from '@uni/unit-tool';
```

你也可以从大包引入：

```js
import {unitTool} from '@uni/apis';
const { px2rpx, rpx2px } = unitTool;
```

## 方法
### `px2rpx(val)`
px 转 rpx 方法。

### `rpx2px(val)`
rpx 转 px 方法。

#### 入参
| 成员 | 类型     | 描述  | 默认值 | 支持  |
| ---- | -------- | ----- | :----: | :---: |
| val  | `number` | px 值 |   -    |       |
