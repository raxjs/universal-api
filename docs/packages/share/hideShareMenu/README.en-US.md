## Install

```bash
$ npm install @uni/share --save
```
or
```bash
$ npm install @uni/apis --save
```
## Usage

```javascript
import { hideShareMenu } from '@uni/share';

hideShareMenu();
```

你也可以从大包引入：
```javascript
import { share } from '@uni/apis';
```
