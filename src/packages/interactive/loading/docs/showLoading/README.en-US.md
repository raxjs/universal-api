## Install

```bash
$ npm install @uni/loading --save
```
or
```bash
$ npm install @uni/apis --save
```
## Usage

```javascript
import {showLoading} from '@uni/loading';

showLoading({
  content: 'loading...'
});
```

You can also import from the big package:

```js
import { loading } from '@uni/apis';

loading.showLoading({
  content: '加载中',
});
```
