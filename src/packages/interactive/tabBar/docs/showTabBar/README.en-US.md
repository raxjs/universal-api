## Install

```bash
$ npm install @uni/tab-bar --save
```
or
```bash
$ npm install @uni/apis --save
```
## Usage

```javascript
import { showTabBar } from '@uni/tab-bar';

showTabBar();
```

You can also import from the big package:
```js
import { tabBar } from '@uni/apis';

tabBar.showTabBar();
```
