## 安装
```bash
$ npm install universal-device --save
```

## 使用

```javascript
import { platform, appName, screenWidth, screenHeight } from 'universal-device';
```

## APIS

### `appName: String`
app 名称或浏览器名称。

### `platform: String`
平台： 'Android'、'iOS'、'MacIntel' 等。

### `screenWidth: Number`
屏幕宽度，单位为px。

### `screenHeight: Number`
屏幕高度，单位为px。
