## Install

```bash
$ npm install universal-unit-tool --save
```
## Usage

```js
import { px2rpx, rpx2px } from 'universal-unit-tool';
```

## APIS
### `px2rpx(val)`
px to rpx method

### `rpx2px(val)`
rpx to px method

#### Arguments
| Property | Type     | Describes | Default | Supports |
| -------- | -------- | --------- | :-----: | :------: |
| val      | `number` | px value  |    -    |          |
