## Installation

```bash
$ npm install miniapp-api-creater --save-dev
```

## Usage

### Set script in package.json
```json
{
  "scripts": {
    "create-api": "create-api -c ./config"
  }
}
```

## Create

```bash
$ npm run create-api
```

### Demo

```bash
$ cd ./demo
$ ../bin/create-api.js -c ./config -d ./build
```
